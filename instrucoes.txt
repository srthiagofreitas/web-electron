Projeto de estudo com EletronJs

Pasta do projeto
mkdir web-eletron

Criando o projeto
npm init -y

Editar no package.json

{
  "name": "your-app",
  "version": "0.1.0",
  "main": "main.js"
}

Criar aquivo main.js
Criar aquivo .gitignore e adicionar node_modules/

Fazer o primeiro commit

Isntalar o Electron
npm install electron
